# Data Directory

This directory stores fetched and processed data files generated by the toolkit.

## Structure

```text
data/
├── dataset/        # Main parquet outputs
├── latest_result/  # CSV previews
├── data_clean/     # Cleaned/derived parquet outputs
└── state.json      # Checkpoint state
```

## Notes

- Runtime data in this directory is ignored by git.
- Keep this directory as a workspace, not as a source-controlled dataset dump.
- For reproducible releases, publish large parquet files to Hugging Face or another data registry.
